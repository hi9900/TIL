## CSS 상속

- CSS는 상속을 통해 부모요소의 속성을 자식에게 상속한다.

- 속성(property) 중에는 상속이 되는 것들과 되지 않는 것들이 있다. 

> ### 상속되는 것 예시

  - Text 관련 요소: font, color, text-align

  - opacity

  - visibility

> ### 상속되지 않는 것 예시

  - Box model 관련 요소: <br>
  width, height, margin, padding, border, box-sizing, display

  - position 관련 요소:<br>
  position, top/right/bottom/left, z-index
