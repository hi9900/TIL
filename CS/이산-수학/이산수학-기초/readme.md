### 1강 이산수학 개요

- 불연속적인 숫자를 다루는 수학, 컴퓨터를 위한 수학

- 이산 수학에서 다루는 내용은 자료구조, 알고리즘 등의 베이스가 되어 전체적인 컴퓨팅 사고력을 길러준다.

- 또한 수학접 귀납법 등의 다양한 기초 개념이 알고리즘에 반복적으로 출현하기 때문에 컴퓨터 과학의 베이스 학문이라고 할 수 있다.
